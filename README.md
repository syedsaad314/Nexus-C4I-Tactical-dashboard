# Nexus C4I: Tactical Command & Intelligence Dashboard

Nexus C4I is a high-fidelity, interactive C4I (Command, Control, Communications, Computers, and Intelligence) simulation dashboard. Built with a focus on real-time data visualization and tactical oversight, it features a futuristic "Deep Midnight" interface and a robust TypeScript-driven backend.

## 🚀 Key Modules
* **Interactive Tactical Map:** A global visualization tool tracking active cyber-threats across different continents using geographical coordinate mapping.
* **Intel Stream Active:** A live, scrolling intelligence feed displaying real-time threat levels, IDs, and regional data.
* **Cyber Analytics Engine:** A dynamic bar chart comparing "Cyber Defenders" vs "Intruders" to monitor system integrity.
* **Modular Command Sidebar:** Quick access to system status, mission timelines, and intelligence reports.

## 🛠 Tech Stack
* **Framework:** Next.js (App Router)
* **Language:** TypeScript (for type-safe tactical data)
* **Styling:** CSS3 with a custom futuristic theme and responsive layouts.
* **Mapping:** React-Simple-Maps for geographical data visualization.
* **Icons:** Lucide-React for a clean, professional UI.

## 📂 Installation & Setup
1. **Clone the repository:**
   `git clone https://github.com/syedsaad314/Nexus-C4I-Tactical-dashboard.git`
2. **Install dependencies:**
   `npm install`
3. **Run the development server:**
   `npm run dev`
4. **View in browser:**
   Open `http://localhost:3000` to see the dashboard in action.

## 📝 Project Vision
This project is a personal exploration into high-fidelity UI/UX and complex dashboard logic. It aims to demonstrate how modern web technologies like Next.js and TypeScript can be used to build crash-proof, interactive interfaces for mission-critical simulations.
